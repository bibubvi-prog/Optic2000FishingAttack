# Optic Fish 🐟

**Poisson d'Avril 2026** – Blague interne à destination de l'équipe RSSI de **Lissac** et **Optic2000**

---

## 🎣 Présentation

Page web humoristique créée pour le **1er avril 2026**.  
Elle met en scène un poisson qui nage à travers l'écran tout en félicitant l'équipe RSSI pour sa vigilance exemplaire.


---

## ✨ Fonctionnalités

- Design full-screen immersif avec dégradé violet/noir
- Animation fluide d’un poisson qui traverse l’écran en continu
- Bulles d’air qui remontent pour l’ambiance "océan numérique"
- Deux emplacements dédiés aux logos **Lissac** et **Optic2000** (en haut de page)
- Message de félicitations personnalisé à l’équipe RSSI
- Effet glow et animations rétro-gaming (police Press Start 2P)
- Interaction : clique sur le poisson pour le faire accélérer
- Clique sur le titre → message surprise "Poisson d'Avril"
- 100% autonome (un seul fichier HTML)


---

## 🚀 Comment utiliser

1. Télécharge le fichier `optic-fish.html`
2. Ouvre-le avec n’importe quel navigateur (double-clic)
3. Profite de la blague et partage-la avec l’équipe !

> **Astuce** : Tu peux aussi l’héberger sur GitHub Pages, Netlify ou tout autre service statique.

---

## 🎨 Personnalisation

Tu peux facilement modifier :
- Les URLs des logos dans la section `.header-logos`
- Les couleurs dans le `<style>` (variables principales : `#ff3366`, `#cc00ff`, `#330033`)
- Le texte de félicitations dans la div `.message`

---

## 🛠 Technologies utilisées

- HTML5
- CSS3 (animations, gradients, flexbox)
- JavaScript vanilla (aucune dépendance externe)
